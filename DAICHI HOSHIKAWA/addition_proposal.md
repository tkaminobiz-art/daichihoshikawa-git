# 追加リンク提案リスト（計14件）

奈良県公式ウェブサイトより、県民生活に直結する**補助金・助成金**を中心とした有用なポータル/制度をリストアップしました。
既存の4件に加え、ご要望のあった「県民が使える補助金や助成金」など10件を追加しています。

## 【守る】医療・健康・セーフティネット
### 1. がんネットなら（奈良県がん情報提供ポータル）
- **URL**: `http://www3.pref.nara.jp/gannet/`
- **内容**: がん検診、患者サロン、相談窓口など県内のがん情報の総まとめサイト。

### 2. 指定難病医療費助成制度
- **URL**: `https://www.pref.nara.jp/dd.aspx?menuid=11607`
- **内容**: 指定難病の治療にかかる医療費の一部を助成する制度。自己負担の上限管理などが受けられます。

### 3. 小児慢性特定疾病医療費助成制度
- **URL**: `https://www.pref.nara.jp/dd.aspx?menuid=11607`
- **内容**: 慢性疾患を抱えるお子さまの医療費負担を軽減する制度。18歳未満が対象です。

### 4. 奈良県ナースセンター（看護職支援）
- **URL**: `http://www.nara-kango.or.jp/nur-01.html`
- **内容**: 看護職の求職・復職支援。医療現場の人手不足解消に向けた重要な窓口です。

### 5. 心身障害者扶養共済制度
- **URL**: `https://www.pref.nara.jp/dd.aspx?menuid=1273`
- **内容**: 障がいのある方を扶養している保護者が、毎月掛金を納めることで、万一の際に年金を支給する任意加入の制度です。

### 6. 生活福祉資金貸付制度
- **URL**: `https://www.pref.nara.jp/1270.htm`
- **内容**: 低所得・高齢・障がい者世帯への生活費等の貸付制度。社会福祉協議会が窓口となるセーフティネットです。

## 【育む】子育て・教育・未来
### 7. なら結婚応援団
- **URL**: `http://www.naradeai.pref.nara.jp/`
- **内容**: 県主催の婚活イベントや結婚支援情報のポータルサイト。少子化対策の入口となります。

### 8. 児童扶養手当
- **URL**: `https://www.pref.nara.jp/1648.htm`
- **内容**: ひとり親家庭等の生活の安定と自立を促進するための手当。

### 9. ひとり親家庭等自立支援給付金
- **URL**: `https://www.pref.nara.jp/dd.aspx?menuid=1272`
- **内容**: 親の資格取得や職業訓練を支援する給付金（高等職業訓練促進給付金など）。

## 【稼ぐ・暮らす】仕事・住まい・移住
### 10. 奈良県住宅リフォーム・耐震改修支援
- **URL**: `https://www.pref.nara.jp/1260.htm`
- **内容**: 既存住宅の耐震改修やバリアフリー化に対する市町村との連携補助情報。

### 11. 県産材を使用した住宅助成事業
- **URL**: `https://www.pref.nara.jp/1260.htm`
- **内容**: 奈良の木を使用した住宅の新築・増改築に対する助成金。

### 12. 職業訓練受講給付金（求職者支援制度）
- **URL**: `https://www.pref.nara.jp/1284.htm`
- **内容**: 雇用保険を受給できない求職者が、職業訓練を受けながら月額10万円の給付を受けられる制度の案内。

### 13. 就職氷河期世代活躍支援
- **URL**: `https://www.pref.nara.jp/1284.htm`
- **内容**: 安定した就労を希望する氷河期世代（30代半ば〜50代半ば）への集中支援プログラム。

### 14. 奥大和移住定住交流センター engawa
- **URL**: `https://okuyamato-engawa.jp/`
- **内容**: 南部・東部への移住相談窓口。空き家バンクや仕事情報の紹介。
